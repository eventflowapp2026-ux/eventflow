Recommended .env File Format
# Email Configuration (Gmail)
MAIL_USERNAME=your.email@gmail.com
MAIL_PASSWORD=your-16-character-app-password  # NOT your regular password
MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_DEFAULT_SENDER=your.email@gmail.com

# App Security
SECRET_KEY=your-secret-key-change-this-in-production

# Important Notes:
# 1. Must use Gmail with 2-Factor Authentication enabled
# 2. Generate App Password from: https://myaccount.google.com/security
# 3. Select "Mail" app and "Other" device
# 4. Copy the 16-character password (no spaces)
Common Issues:
  • Using regular password instead of App Password
  • 2-Factor Authentication not enabled
  • Quotes around email in .env file (should be: email@gmail.com, NOT "email@gmail.com")
  • Network blocking port 587
