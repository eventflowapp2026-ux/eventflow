# Event Registration System

A comprehensive event registration system built with Flask that allows users to create events, build custom registration forms, and collect responses.

## Features

- **User Authentication**: Sign up and login with email/mobile number
- **Event Management**: Create events with descriptions and images
- **Form Builder**: Create custom registration forms with various question types:
  - Text, Number, Email, Date
  - Multiple Choice, Checkboxes, Dropdown
  - File Upload
  - Long text areas
- **Response Collection**: Collect form responses via shareable links or QR codes
- **Data Export**: Download responses as CSV or PDF
- **Dashboard**: Manage all events and forms from one place
